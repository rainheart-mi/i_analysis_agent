Ant Design X 整体布局方案

现有页面是 3 栏：左=任务实例 / 中=节点画布 / 右=ChatPanel（360px）。下面对比 3 种集成方式。

A

右栏原地升级（最小侵入）

保持 3 栏结构，仅把 ChatPanel 内部组件换成 Bubble / Sender / 预设 Prompts。其它面板零改动。

B

新独立全屏对话页（新路由 /chat）

在 React Router 加 `/chat` 路由，AI 助手页加按钮跳转。独立布局：左 Conversations + 中 Bubble + 底 Sender。沉浸感最强。

✅ 沉浸、宽屏、虚拟列表 ❌ 跳出当前上下文；与 NodeExecution 绑定变隐式

C

右栏 + 抽屉展开双形态（推荐）

默认 360px 紧凑右栏（用 Bubble / Sender），右上角"展开"按钮 → 抽屉式全屏对话（带 Conversations 多会话切换）。一个组件代码，两个模式。
